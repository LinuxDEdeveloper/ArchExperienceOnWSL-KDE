# ArchExperienceOnWSL-KDE
## Experience Archlinux on WSL2 with KDE Plasma 6 with almost full functionality and more!

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

# Feature:
## + Linux-based operating systems such Arch Linux can be used directly and familiarized with before being installed directly on a computer, instead of using WSL after installed on a real computer. (Of course)
## + Familiar with the fast-updating (Rolling Release) linux distros in particular and Linux distros in general.

# Disadvantages
## + It's difficult to get used to for people who are afraid of using WSL and Terminal, those who know nothing about Arch Linux, basic commands on Linux and Powershell,edit feature on BIOS (VT-x or AMD-v), and operations from Windows.
